## Qué es Pycantic

**Pydantic** es una librería de validación para Python. Es rápido y
extensible, se integra con *linters*, IDE. Permite definir como deberían
ser los datos en Pure Python 3.9+. Se puede validar con Pedantic, y se
puede monitorizar con *Pydantic Logfire*.
