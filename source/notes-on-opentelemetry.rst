¿Qué es OpenTelemetry
-----------------------------------------------------------------------

**OpenTelemetry** es una colección de APIs, SDKs, y herramientas, que se
pueden usar para generar, recolectar y/o exportar datos de
`telemetría`_ (como métricas, *logs* y trazas de ejecución) que se pueden
utilizar para analizar y mejorar el comportamiento y el rendimiento de
nuestro *software*.
